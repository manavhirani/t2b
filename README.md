# t2b
## A text to binary (ascii) converter written in python
Designed as a simple command line tool

### Requirements
- Have a local installation of python 3+

### Usage
- Navigate to the root directory and run
```bash
    $ python t2b.py <text0> <text1> ... <textn>
```
- Press -h for usage
